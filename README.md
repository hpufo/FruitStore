# FruitStore
A fruit store built using React and Redux. The app get's it's stock from a json file.

Technologies used:
React,
Redux,
Redux-thunk,
Axios,
Webpack,
CSJS(For CSS in JS)

Useful commands:

"npm/yarn install" to install dependencies 

"npm/yarn start" or "npm/yarn run dev" to run dev server on http://localhost:8080/
